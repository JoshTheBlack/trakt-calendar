The original key is lost

Only if the original key is truly gone. This permanently discards every value the current key cannot open — you will re-enter the API keys in Settings, and each user will re-link Trakt. Accounts and their other data are kept; only the unreadable tokens are cleared. Values that DO open under the current key are left untouched.
